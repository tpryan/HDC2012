The Users are Restless

Kevin Hoyt and Terry Ryan

After years of relative peace and calm following the Browser Wars, web workers got complacent. Now a slew of modern tools and techniques demonstrate to our users that the web can be better, it can work on mobile, and it doesn't have to be in the browser. They have seen and voted with their clicks. Adopting modern web design techniques like Responsive Web Design is an imperative. Looking to time tested solutions like Semantic HTML will save us effort and time, if we apply it. We'll talk about where we are coming up short, and the hows and whys of delighting our users again.
